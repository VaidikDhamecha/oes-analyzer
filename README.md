# OES Analyzer (Online Examination System)

A web-based platform for conducting online examinations with real-time performance analytics and detailed post-exam solution reviews.

## Key Features
- **Intuitive Exam Interface:** Secure and user-friendly test-taking environment.
- **Performance Analytics:** Dashboards for tracking progress and identifying knowledge gaps.
- **Solution Review:** Detailed post-exam feedback showing user choices vs. correct answers.

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL
